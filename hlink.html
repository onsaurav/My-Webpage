Link

Today web page is the best way to comunicate with the world. But it is possibal for linking. Without link we can't think about a webpage. The main tag of link is < A >. Which called anchor tag. this tag has HERF attributes. HERF contains the name of linking parts. < /A > is the closing of this tag. What we write inside of this two tags that will show in the docoment. We devide the links in two types.

  • Internal Links ( Link what use for Internal linking )
  • External Links ( Link what use for External linking )

Internal Link

In ternal means link in same site. When we create a link in the corrosponding page or with another page of this site then its called enternal link. It also has some parts. Link with another page what is in this folder, Link with a page what are not in this folder, Link in the same page. In same page link, we declar a name then create a link with that.

Link with a page of same folder < A HERF = " file name . extention " > link text what show < /A >
Link with a page what are in different page. < A HERF = " folder name\file name . extention " > Link text < /A >
Link in the same page. < A NAME = " #Name " > Name of paragraph what show < /A >
< A NAME = " Name " > Name of link what show < /A >

External Link

External link is that where we create a link with anothere site or something what is not related with it. To link with a web-site or mail number we use this type of link.

Link with Mail Number < A HERF=" mailto : sauravcs@yahoo.com " > Name of the link < /a >
Link with Web-site < A HERF=" http://www.Sworld.com " > Name of the link < /a >

Color of link, active link, vesited link can be defined by a designer with body tag. like < Body link = " blue " alink = " pink " vlink = " green " >. Link contains the value of link color. Alink contains the color of active link and vlink contains the color of vesited link.

Link with an Image

Like the text we can also use an image to link. If we click on that image then it shows the linking page. It is not so hard but as like as previous link. Suppose we want to link with a page which name is Cricket.html. Then the code will be

< A HERF = " Cricket.html " > Image file name . Extention < /A >

Different parts of an image can create link with different pages (Maping).

Now we descouse a process which allows us to create a different links by the different place of same image. To create this we need a special tag < MAP >. At first we give the name of the map. Then we the image in some area coords. We create link with this area coords. The followings table shows the code of this maping.

< MAP NAME = "Mapname ">
< AREA COORDS = " 0,0,75,50 " HREF = " Intro.html " SHAPE = " Rect " ALT = " Introduction " >
< AREA COORDS = " 0,100,50 " HREF = " Backg.html " SHAPE = " Circ " ALT = " Background " >
< /MAP >
< IMG SRC = " File name . extention " width = "350" height = "400" USEMAP = " #Mapname " BORDER = " 0 " />

Mapname : Use to give the map name. It can be a, b, x, y, Saharukh, Aisharya etc. When we add the image then that image use this mapname.

Area Coords : It declar to devide the image area in to some parts. we see in the coading ( 0, 0, 75, 50 ), ( 0, 100, 50 ). The area devide basis of that values. Four degites for rectangle and three degites for circle.

Shape : It is the shape of linked area. It contains the value like RECT, CRIC etc. When you click on that link then you will see that shape.

Alt : When you visite on the image if you reached on the linked area then you see that text what you write in alt.
