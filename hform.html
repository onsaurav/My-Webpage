The basic tag of creating a form is < FORM & gt < /FORM >. This form tag has some attributes like action, method, enctype, name, target etc.

Action : It is the URL of the CGI( Common Geteway Interface ) program that is going to accept the data from the form, process it, and send a response back to the browser.

Method : GET (default) or POST specifies which HTTP method will be used to send the form's contents to the web server. The CGI application should be written to accept the data from either method.

Enctype : detrmines the mechanism used to encode the form contents. you can live this attribute as default unless you are creating a file upload field.

Name : is a form name used by VB or Java script.

Target : is the target frame where the response page will show up.

Form elements have properties : text boxes, Password boxes, Checkboxes, option ( Radio ) buttons, Submit, Reset, File, Hidden and Image. This properties are specified in the TYPE attribute of the HTML element < INPUT > < /INPUT >.

< INPUT > ELEMEN'S PROPERTIES
TYPE = Type of input
Name = Variable name passed to CGI application
VALUE = The data associated with the variable name to be passed to the CGI application
CHECKED = Button/box checked by default
SIZE = Number of characters in text field
MAXLENGTH = Maximum number of characters accepted


All the attributer use inside of < form action = " " Method = " "> < /form >
