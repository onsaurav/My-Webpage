Margin:

CSS aims to give you total control over margins, but not only for the body, but also for any block elements: lists, images, paragraphs, div's, span's, etc. Margin values can be fixed amounts, such as px (pixels), or % (a percentage), or set to auto. Margins work clock-wise.
If you want a page to have no margin, you could use: You Can give margin only one or two side. Then the value of other two side will be zero. The values will be in px or %.

body { margin: 0px 0px 0px 0px; }
body { margin: 25px 15px 10px 10px; }
body { margin: 10px 0px 0px 0px; }
body { margin: 10% 0px 0px 0px; }

Padding:

Padding: is just like a margin, except it's the white space between the margin (and border) and the actual content. You can also use auto or a % (percentage), just like a margin. The four individual padding properties are padding-top, padding-right, padding-bottom, and padding-left.

p { padding: 5px 5px 5px 5px; }

Border:

A border can be a visual separator between the margin on the outside, and the padding on the inside. The value can be
thin, medium, thick, or a numeric value and unit, such as 5px or 0.2em.

p { border-width: medium; }
p { border-width: medium medium; }
p { border-width: medium medium medium medium; }

The values for border-style are:
none, dotted, dashed, solid, double, groove, ridge,inset, outset.

p { border-width: medium; border-style: dotted; }

Border-color property. Just set it to blue:

p { border: medium dotted blue; }

All of them are together.

p { border-width: medium; border-style: dotted; border-color: blue; }

For individually borders the properties are border-top, border-right, border-bottom, and border-left.

p { border-left: dotted red; border-right: dashed purple; }

Filter:

To make a good CSS you may need to use filters. For texts you have glow(), shadow(), dropshadow(), wave(), blur() filters. Example with a class.

h1.x{ width : 100%;
filter : glow(color=blue,strength=1 to 10); }

h1.x{ width : 100%;
filter : blur(strength=1 to 10); }

h1.x{ width : 100%;
filter : shadow(color=green); }

h1.x{ width : 100%;
filter : dropshadow(color=lime); }

h1.x{ width : 100%;
filter : wave(strength=1 to 10; }

Now apply to h1

<h1 class="x">Checking Filter </h1>

For images you have gray(), xray(), invert(), alpha(), chroma() filters.

img.x{ width : 100; height : 100; filter : gray(); }

img.x{ width : 100; height : 100; filter : xray(); }

img.x { width : 100; height : 100; filter : invert(); }

img.x{ width : 100; height : 100; filter : alpha(opacity=0 to 100); }

img.x{ width : 100; height : 100; filter : chroma(color=#000000); }

Now apply to Image

< img class="x" src="synergy.jpg" width="240" height="180">
