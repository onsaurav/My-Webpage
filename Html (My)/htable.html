Table

Table is a very very essential components of web-page. It is use not only for create a table of data. It is also use to design a page. By this table we can design a paralal document with information and image. The main tag of table is < Table > and < /Table >. < Tr > contains the value for table row. < Td > contains the value for table data. < Th > is use to show table header. And some attrebutes are also used with table thats are given in a table.

Attributer Description
Border Contains the value of border like 1,2 etc. If it absence then no border will show.
Bgcolor Contains the value of Background color likr color name or hexa decimal code.
Align The value of alignment. Like Center, Left or Right.
Width Contain the width of the table in pixel
Cell padding the space between the content and inside edge of a cell. Values possible are 1, 2, 3, and so on.
Cell spacing the amount of space between cells in a table. value contains 1, 2, 3 and so on.
< Caption > It is a tag what use inside of table. Like < Caption > and < /Caption >. It is use to table title or Caption. it stands inside of the table but not inside of the row or cells. Its align value Top and Buttom

Some attributes what is use with < TR >

Bgcolor to create a background color
Colspan Width of the cell in turms of number of columns. Use when a cell occupies more then one colomn.
Rowspan Width of the cell in turms of number of rows. Use when a cell occupies more then one row.

Table data ( < Td > ) also contains the attributes like < Tr >. A code of table and a result are given below. The code will be < Table Border="1" Cell spacing="5" Align="center" > and for table row (tr) < Tr Bgcolor="RED" colSpan="3" >. Code of table data are also like this. < td width="200" >

< Table Border="2" Align="center" cell spacing="5" >
< Tr Bgcolor="#336699" >
< Td Rowspan="3" > < /Td >
< Td width="310"> < /Td >
< /Tr >
< /Table >
