List

List is an important items of web page designing. If we Browse a site then we must meet with list there. List is basicaly a set of related item. Three type of list we get heat. Unorderd List, Ordered List, Defination List. List also can be nested type. All the types are follows.

Unordered List

Ul ( Unordered list ) : Items in this list start with a list mark such as a bullet. Browsers will usually change the list marks in nested lists. Li is the list item.Ul has three basic types. Disc is default. Other two is Circlr and Square.

< UL TYPE = "SQUARE" >
< Li >First Item < /Li >
< Li >Second Item < /Li >
< Li >Third item < /Li >
< /UL >
  • First Item
  • Second Item
  • Third Item

Ordered List

Ol, ( Ordered list ) : Items in this list are numbered automaticaly by the browser. You have the choice of setting TYPE attribute to one of five numbering style. Those are 1 ( 1, 2, 3, 4, 5, ....... ), a ( a, b, c, d, e, ........ ), A ( A, B, C, D, E, ... ), i ( i, ii, iii, iv, v, ......... ), I ( I, II, III, IV, V, ... ). Another attributes START is also use with ordered list. It contains the value of starting.

< OL TYPE = "1" START="5" >
< Li >First Item < /Li >
< Li >Second Item < /Li >
< Li >Third item < /Li >
< /OL >
  1. First Item
  2. Second Item
  3. Third Item

Defination List

DL, ( Defination List ) : This kind of list is different from the others. Each item in a DL consists of one or more Deffination terms (DT) followed by one or more Defination descriptions (DD).

< DL >
< DT > HTML: < /DT >
< DD > Hyper Text Markup Language < /DD >
< DT > CSS: < /DT >
< DD > Cascading Style Sheet < /DD >
< /DL >
HTML:
Hyper text Markup Language
CSS:
Cascading Style Sheet

Nested List

Nested list means list in a list. If we use a list in a list item then a nested list will created. All types of list ( UL, OL, DL ) can take a part in a nested list.

< Ul TYPE="circle">
 < li > Programming : < /li >
    < Ol TYPE="1">
       < li > C < /li >
       < li > C++ < /li >
       < li > Visual Basic < /li >
            < OL TYPE = "a" >
                 < li > VB < /li >
                 < li > VB.net < /li >
            < /OL >
    < /Ol >

    < /Ol >
 < li > Graphics < /li >
    < Ol TYPE="i">
       < li > Photoshop < /li >

       < li > Flash < /li >
    < /Ol >
< /Ul >
