Sound and Movies

We can use movie (vedio), animation or audio file in our page. But that types of file is large type. So we need extra care to use that. Becouse if our page will so large then it create a probleme to load. what wast our time. So some format can easily loaded in the document like .AVI, .MPEG, .QT etc. Animation and Flash movie also added with a page.

Suppose we want to a video and an audio file in our site. the name of vedio file is (f.mpeg) and the name of the audio file is (o.mp3). then the code is:

< EMBED SRC="f.mpeg/o.mp3" WIDTH="250"HEIGHT="200"AUTOSTART="true"SHOWCONTROLS="true" ALIGN="left">
OR
< IMG BORDER="0" DYNSRC="f.mpeg/o.mp3" START="fileopen" WIDTH ="250" HEIGHT="200" >

EMBED : Used to add animation or vedio or audio file on the page.
SRC : Detecting the source of the file.
WIDTH : Contains the value of the file width.
HEIGHT : Contains the value of the file height.
AUTOSTART : If its value is true then the file will start when you open this page.
SHOWCONTROL : Use to showing control in the page.
ALIGN : Used to select file alignment.

IMG BORDER: Use to add file and select border value like 0,1, 2 etc.
DYNSCR : It indecate the dynamic path of the file. Hear youdeclar the file name with extention.
START : It is used to start the file. it value like "fileopen".

Animation movie are also add in this way. Suppose we want to add a flash movie then We dont use SHOWCONTROL attribute. The code will be:

< EMBED SRC = "file.extention" WIDTH ="150"height="200" AUTOSTART="true">
