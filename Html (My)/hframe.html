Frame

A Framed psge is actually made up of multiple HTML pages. There is one HTML Document that describe how to break up the single browser window into multiple windowspan. Each windowspan is filled with an HTML document.

It is a picture which shows a framed page. Where the page contains three pages. To create a framed page we use the tag < FRAMESET >. This tag is placed in the HTML document before the < BODY > elements. The < FRAMESET > describe the amount of screen real estate given to each windspan by deviding the screen into ROWS or CILS. The < FRAMESET > will then contain < FRAME > elements, one per division of the browser window. The code of what use to create a framed page are follows,

< HTML >
< HEAD >
< TITLE >Framed Page < /TITLE >
< /HEAD >
< FRAMESET CILS = "20%,80% " >
< FRAME SRC="doc1.html" NAME="first" SCROLLING ="no" >
< FRAMESET ROWS = "15,* " >
< FRAME SRC="doc2.html" NAME="second" SCROLLING ="no" >
< FRAME SRC="doc3.html" NAME="third" SCROLLING ="yes" >
< /FRAMESET >
< NOFRAMES >
< BODY >
<!--Document for browsers that do not support framed-->
< /BODY >
< /NOFRAMES >
< /FRAMESET >
< /HTML >

Attributes of < FRAMESET >

Frameborder : Possible values 0,1, YES, NO. A setting of zero will create a borderless frame.
Framespacing : Create space between two frame. It contains the values in pixel. Value zero declar no space between frames.
Border : Possible values 0,1,. A setting of zero will create a borderless frame.
Bordercolor : This attributes is allows uoy choose a color for frame border. This attribute is realy used.

Attributes of < FRAME >

SRC : Required, as it provides the URL for the page that will be displayed in the frame.
Name : Required for frame name that allow targeting by other HTML documents.
Scrolling : Displays a scroll bar(s) in the frame. Possibal values are yes (Always display scroll bars), no (Never display scroll bars), auto (browser will decide based on frame contents). The default value is auto.
Marginwidth : Optional attribute stated in pixel. Determines horizontal space between the < FRAME > contents and the frame borders.
Marginheight : Optional attribute stated in pixels. Determines vertical space between the < FRAME > contents and the frame's border.
Noresize : optional - prevents viewers from resizing the frame.

TARGETS

When you are creating links for use in a frames environment you will need to specify an additional attribute called TARGET. The TARGET attribute uses the NAME attribute of the FRAME element.The code to create link in framed page < A HREF = "page-name.extention" TARGET="frame-name " >What show in browser < /A >. If we fail to declar target then page will open in current page. The frame name and target is case sencative. The TARGET has followings elements.

< A > : Sample - < A HREF ="file.html" TARGET ="frame -window-name" >
< AREA > : Sample - < AREA SHAPE ="rect" COORDS="5,6,2,4" HREF ="file.html" TARGET ="Frame-window-name" >
< BASE > : Sample - < BASE TARGET ="Frame-window-name" >
< FORM > : Sample - < FORM ACTION = "cgi-bin/script" TARGET = "Frame-window-name" >

Special Target

There are four types of special target names that cannot be assigned by the name attribute of the FRAME tag. Each of these reserved names serves a special function when used with the TARGET attribute. These special names and functions are:

TARGET="_top" : This loads the linked gocument into the full browser window with the URL specified by the HREF attribute. This is particulary useful for moving between a frames enviorment ant non-frames environment.
TARGET="_blank" : Opens a new browser window and loads the document specified in the URL attribute into that new window. The window is not named.
TARGET="_self" : Loads the document in the same window where the anchor was {CLICKED}. This is the default setting for linking elements.
Target="_parent" : The _parent frame is a prior frameset that the current frameset was "spawned" from. If there isn't one then it is the browser window.

Code of Special Target
< A HREF ="doc.html" TARGET="_top" > Document in top < /A >
< A HREF ="doc.html" TARGET="_blank" > Document in blank < /A >
< A HREF ="doc.html" TARGET="_self" > Document in self < /A >
< A HREF ="doc.html" TARGET="_top" > Document in parent < /A >
< A HREF ="doc.html" TARGET="_parent" > Document in parent < /A >
