Background

A normal page is not intersting for all time. Some times some images or color are essential to describe the corrosponding text or document. Some time the document totaly depend on image or color. So for that type of pages we need Background image or Color or some imageses to add with the page.

Background : Baackground is that what stay on the back side of the page. Total document stay on it. We can use Corrosponding image or Color as a background.The code of background are given below. It must be wretten with the body tags. Becouse we descouse about body Background

< body background ="File name . extention" >
< body background ="olive" >
< body background ="#336699" >
< body bgcolor ="red" >
< body bgcolor ="#FF0000" >

Previous four lines is the code of background. In the first line we want to add an Image as a background. So we use the file name with extention. In second line we want a color as background. So we use the name of color(olive). This job can be complete in another way. Where we use the color code like line number three. In fourth and fifth line we use bgcolor. It is same to background. But you can't use it to get an image as a background.

IMAGE

When a page is designed then image is not only used for background. Actualy image used every where of the page. So we need to know the use of image. We can use the image by using followings code.

< IMAGE SRC = " file name.extention " width = " 100 "height = " 250 "border = " 1 ">

Image tag has some parts. All the parts can used together. One can use only one or to from it. (If the image are is in this folder then the image file name .Extention is enough or if the image is in different folder then the correct path will be needed. Suppose the file name is pic.jpg and it is in same folder then pic.jpg is enough. If the file stays in a folder S of G:\ then the code will be < img src = " g:\s\pic.jpg " > ) Width contains the value of the width of the picture. Height contains the value of the height of image. Border contains the value of the border. Align tag is also use with it. So the picture can be used in different alignment. The Height and Width propertis contains the value in pixel. And align contains the value like top, left, right, center, Texttop, Baseline, ABCMiddle etc. HSPACE and VSPACE is aditional image attributes. HSPACE is use to maintain the Horizontal space and VSPACE maintance the vartical space.

But some format of image are not supporetd by it. So, if we want to use that type of image then it doesn't show on the page. Supporting formats are GIF, JPG, JPGE, BMP etc. And the non-Supporting formats are PNG, TIFT, PFT etc.
