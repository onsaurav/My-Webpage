Some Extra Effect

Till now we learn about text and image to create a page. But all of the text and images are still. Some where you should need to move the text. Or some time you fell that some extra effect is need to creat best output. Then what can you do? HTML has some tags what used for that tipe of scrolment or movement of text. We descuse that follow.

SCROLL

SCROLL (R)

SLIDE

ALTERNATE

In this page we see some movement of text. < MARQUEE > tag is use for this scrollment. The first text is scroll from left. Second is scroll from right. Third one is slide type. Last one move from one side to another. It is alternative. The code of movment are follows.

Code of Movement
< MARQUEE DIRECTION ="left" BEHAVIOR ="scroll" SCROLLMENT ="5" SCROLLDELAY ="3" > SCROLL < MARQUEE >
< MARQUEE DIRECTION ="right" BEHAVIOR ="scroll" SCROLLMENT ="5" SCROLLDELAY ="3" > SCROLL (R) < MARQUEE >
< MARQUEE DIRECTION ="left" BEHAVIOR ="slide" SCROLLMENT ="5" SCROLLDELAY ="3" > SLIDE < MARQUEE >
< MARQUEE DIRECTION ="left" BEHAVIOR ="alternate" SCROLLMENT ="5" SCROLLDELAY ="3" > ALTERNATE < MARQUEE >

Marquee : This tag is use to create this type of text movement.
Direction : Contains the value of derection of movement. Such as Left, right etc.
Behavior : This tag is very useful elements contains the value of behavior. Its value is scroll, slide, alternate etc.
Scrollment : It contains the value of scrolment in single time.
Scrolldelay : Delay between scrollment, contains value like 1, 2 etc.
