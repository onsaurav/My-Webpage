Text

Basicaly a page is the collection of corresponding information. One make a site to give the information of his topics. So, the main part of a page is text. Document can't complete without text. For this reason text and its propertis are very essential. We use the text what has some basic attributes like Font size, Font face, Font color etc.

Some things what we write inside of the < body > tags are shown in the page. But we can give some special things with it. To change the text property we use the < font > tag.

< font face="The New roman" color="Red" size ="5" >Write something hear. < /font >

Font tag has three part. you can use one, two or all the parts. Face contains the name of the fonts like Monotype Corsiva, Impact, Comic Sans MS, Arial, Times New Roman etc. Color contains the value/name of color of the text. Color is used both name and hexa code form. Size contains the Value of size of the text. Size will be in 1 to 7.


PRAGRAPH

Pragraph is a very essential tag. When we read a book then we see that all the documents are devide into some paragraph. Like this we can devide our document into some paragraph with the tag < p >. We can align the text of paragraph by using align with p.

< P > Write the text hear! < /p >
< P align = "Center" > Write the text hear! < /p >

< PRE > is also a tag of HTML. It is as like as Pragraph. But the basic difference is it shows the code as like as you write in the source. If you give a break or space in the source then it will shows like that in the page.

< PRE > Put the text hear!< / PRE >.
Suppose we write the code like

ALIGNMENT

We can show the text in different align by using the align. It has three values like Left, Center, Right. The align tag is use with <p>, < hr/ >, < h1 > etc tags. The code to use align

< p align = " Center " > put teh text hear! < /p >
< hr align = " Left " / >
< h1 align = " Right " > put teh text hear! < /h1 >

When we use the value Center then everythingss will be center alignment. Right value use to right alignment and left value shows the right alignment.
