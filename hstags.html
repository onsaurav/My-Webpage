Special Tags

Some tag we use in a page what creates it more easy to show the corrosponding information. If we need to give the heading of a paragraph then we can do it by incressing the font size. But it is not a good process. We also can do it by a tag < h1 >. It use to give the heading. It has different size what we can express by < h2 >, < h3 > etc. The code and its result is as follows,

Code Result
< H1 > It is in H1 < / H1 >

It is in H1
< H2 > It is in H2 < / H2 >

It is in H2
< H3 > It is in H3 < / H3 >

It is in H3
< H4 > It is in H4 < / H4 >

It is in H4
< H5 > It is in H5 < / H5 >
It is in H5
< H6 > It is in H6 < / H6 >
It is in H6
< H1 align="center" > It is in H1 < / H1 >

It is in H1

We can also use the alignment with this tags. Like < H1 align="center" > Put The Text Hear < /H1 >. Then the heading will show in center. If we use left or right alignment then it will show in that alignment.
