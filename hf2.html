Do you think that "how do i start?" or "What type of software is needed?" .Suppose you have a computer then nothing is needed to start Web page designing. Open the Notepad and write the code of HTML (what you learn next). Save the code with html or htm extention. When you open the file what is created with html/htm extention, you can see your page. HTML is not a case sensative language so you can use either lower or UPPER or Both. The most important thing is extention must be html or htm.

The first tag of HTML is < html >. Look all the tag must be coated by < and >. To finish the execution of a tag suppose < html > we must use < /html >. So, <> it is the start of tag and it is the end of tag. After the < html > tag < head > tag is used. < title > tag is used in < head > tag. Title tag contains the title of the page.after ending the < head > tag < body > tag starts. It is the tag where you write all the equipments of the page. So the normal forms of a html pages is as follows.

< html >
< head > < title > Write the title! < /title > < /head >
< body >

Put the documents for body!   (write anything all are shown in the page.)

< /body >
< /html >

This three tags < html >, < head > and < body > are the most common and essenial of any page. To create a page this tags must be needed.
